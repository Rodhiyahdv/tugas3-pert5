### Pertanyaan
1. Apa perbedaan dari procedural, conditional, dan looping?
2. Kapan harus menggunakan looping?
3. Sebutkan 3 looping yang dapat digunakan dan fungsi masing-masing looping!

### Jawaban
1. Procedural => dilakukan dengan memberikan serangkaian perintah yang berurutan.
   Conditional => suatu opsi dengan kondisi tertentu. Jika kondisi terpenuhi, maka opsi dalam kondisi tersebut akan dijalankan.
   Looping => sebuah urutan perintah yang secara terus menerus diulang hingga suatu kondisi tercapai.
2. Looping harus digunakan ketika memiliki data dalam jumlah besar namun memiliki pola dan penyelesaian yang sama.
3. a. For   => membuat perulangan dimana jumlah perulangannya sudah jelas (diketahui).
   b. While => membuat perulangan berdasarkan suatu kondisi dimana pemeriksaan dilakukan diawal bagian pengulangan.
   c. Do While => membuat perulangan berdasarkan suatu kondisi dimana pemeriksaan dilakukan dibelakang bagian pengulangan. Perulangan akan dilakukan sebanyak 1 kali terlebih dahulu sebelum melakukan pemeriksaan.
